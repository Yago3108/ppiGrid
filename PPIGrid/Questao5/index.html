HTML 5


HTML, ou Hypertext Markup Language, é a linguagem de marcação padrão para a criação de páginas da web, usada para estruturar e apresentar conteúdo através de um navegador web

MAIS DETALHES

Elementos básicos do HTML 5


  • DIV


    Elemento que pode
    guardar outros elementos

  • P


    Elemento para textos
    em HTML

  • H1


    Elemento para títulos que
    se extende para h2,h3...

  • UL


    Elemento para criar
    listas não ordenadas em HTML

  • LI


    Elemento que é colocado
    como item da lista

  • OL


    Elemento para criar
    listas ordenadas em HTML

CSS 3


CSS,ou Cascading Style Sheets é uma linguagem de folha de estilo usada para apresentar documentos escritos em HTML ou XML, permitindo controlar a aparência, o layout e o design de páginas web, como cores, fontes, tamanhos e a disposição de elementos na tela, papel ou em outros meios

MAIS DETALHES

Características básicas CSS 3


  • Background


    Característica que
    altera o fundo do elemento

  • Padding


    Característica que
    altera o espaçamento dentro
    do elemento

  • Margin


    Característica que
    altera o espaçamento
    do elemento no body

  • Color


    Característica que
    altera a cor dos elemtos

  • Display


    Característica que
    altera a forma que o elemento
    é mostrado

  • Align-items


    Característica que
    alinha os itens do elemento
